# CodeAlpha_internship
.
This repository contains two web-based projects developed during my Web Development Internship at CodeAlpha. These projects demonstrate core front-end skills including responsive UI design, DOM manipulation, and logic implementation using HTML5, CSS3, and JavaScript.

📁 Projects Overview
Project 1: Standard Web Calculator

Project 2: Interactive Music Player
🔢 1. Standard Web Calculator
A sleek, responsive calculator capable of performing basic arithmetic operations with a clean user interface.

✨ Features
Basic Arithmetic: Addition, subtraction, multiplication, and division.

Clear & Delete: Functionality to clear the entire screen (AC) or delete the last character (DEL).

Responsive Design: Works seamlessly on desktops, tablets, and smartphones.

Glassmorphism UI: Modern design with subtle shadows and hover effects.

🛠️ Tech Stack
HTML5: Structure and layout.

CSS3: Custom styling and layout (Flexbox/Grid).

JavaScript: Logic for calculations and event handling.

🎵 2. Interactive Music Player
A fully functional music player that allows users to play, pause, and navigate through a playlist of songs.

✨ Features
Play/Pause/Skip: Standard playback controls.

Progress Bar: Interactive seek bar to track and change the song position.

Dynamic UI: Displays song titles, artist names, and album art.

Volume Control: Adjustable volume slider.

Playlist Management: Smooth transitions between different tracks.

🛠️ Tech Stack
HTML5: Semantic structure and Audio API tags.

CSS3: Animations, transitions, and styled controls.

JavaScript: Audio object manipulation, time tracking, and dynamic UI updates.
